Game Name: Bombarded
Description: Top-down space shooter. Enemy tries to crash into the player while the player(s) must survive and try obtain the highest score. Each kill rewards 10 points.
If the player is hit by an enemy spaceship then the player dies. Game ends when there are no more players. 

Controls:
Player 1 movement = arrow keys
Player 1 shoot = right alt
Player 2 movement = W,A,D keys
Player 2 shoot = spacebar
To exit to main menu = ESC

Important Note:
Please load Unity project on Unity version 4.6.x or higher for all features to work.
